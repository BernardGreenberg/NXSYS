This is the code for the fictitious Progman St. interlocking in NXSYS version 2.
To use this, open NXSYS Version 2 (or NXSYS/Mac), invoke "File!Open" from the menu,
and select "progman.trk" in this directory.  Then see "NXSYS Help" in the help menu,
Section 4.1, "Progman St. Interlocking."
