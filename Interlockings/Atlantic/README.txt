This is the source/expr code for Atlantic Avenue interlocking in NXSYS version 2.
To use this, open NXSYS Version 2 (or NXSYS/Mac), invoke "File!Open" from the menu,
and select "atlantic.trk" in this directory.  Then see "Atlantic Avenue Interlocking Info"
in the help menu to learn more.
